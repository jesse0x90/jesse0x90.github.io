Spring Boot学习 -- 创建项目

Posted on 2019-05-03 | Edited on 2019-07-10 | In Spring-Boot Java

0. 前言

最近学习Java后端开发，在学习Spring Boot时，网上的教程大都是通过从Spring官网下载或在IDE里创建或使用Spring Boot Cli的方式来初始化项目，这里总结一下除这两种方式外手动创建项目的方法。

1. Maven创建项目

1.1 使用maven骨架工程生成项目

1.2 pom.xml配置

在pom.xml中添加Spring Boot启动父依赖和web依赖

1.3 编写并运行

编写项目代码，命令 mvn spring-boot:run 运行项目

2. 先创建pom.xml

2.1 创建pom.xml

在项目空的根目录下创建前面的pom.xml文件，使用 mvn dependency:tree 查看依赖，同时该命令会下载相关依赖

2.2 创建项目

创建目录 src/main/java 及其他目录和文件，执行 mvn spring-boot:run 运行项目

3. 附：设置maven镜像

以网易镜像为例，修改maven配置文件即可 /usr/local/Cellar/maven/3.6.0/libexec/conf/settings.xml
